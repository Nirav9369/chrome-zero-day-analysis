# 🔐 Chrome Zero-Day Vulnerability Analysis

## 📌 Overview
This project analyzes two critical zero-day vulnerabilities in Google Chrome:

- CVE-2026-3909 (Skia - Out-of-Bounds Write)
- CVE-2026-3910 (V8 Engine - Code Execution)

These vulnerabilities were actively exploited in the wild and demonstrate real-world browser attack techniques.

---

## 🧠 Key Concepts Covered
- Memory Corruption
- JavaScript Engine Exploitation
- Client-Side Attacks
- Exploit Chaining
- Browser Security Architecture

---

## ⚔️ Attack Flow
1. User visits malicious website  
2. Skia vulnerability triggers memory corruption  
3. V8 engine vulnerability enables code execution  
4. Exploit chaining leads to browser compromise  

---

## 🛡️ Mitigation
- Keep browsers updated  
- Avoid untrusted websites  
- Use sandboxing and endpoint protection

This project demonstrates practical understanding of modern web exploitation techniques and zero-day vulnerabilities.
